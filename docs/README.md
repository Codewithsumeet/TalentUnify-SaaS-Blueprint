# Documentation Layout

This repository's documentation has been organized into focused folders:

- `blueprint/`: Core blueprint and architecture documents (`00` to `10`, index, and correction docs).
- `guides/`: Integration, implementation, UX, and execution guides.
- `research/`: Working notes, exploratory research, and ideation docs.
- `assets/prototypes/`: HTML mockups and visual prototypes.
- `assets/source-material/`: External reference files (`.docx`, `.pdf`).

Runtime code and app folders remain unchanged at the repository root:

- `backend/`
- `talentflow/`
- `tests/`
